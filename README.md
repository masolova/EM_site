# Сайт Елены Масоловой — макеты (v12)

Чистый HTML + CSS, без сборки. Можно открыть локально или залить папку на
GitHub Pages.

## Что изменилось в v12 (по сравнению с v11)

- URL-ы переведены с транслита на английские:
  - `stati/` → `articles/`
  - `tag/` → `tags/`
- Пример статьи получил английский slug:
  `stati/konechnye-i-beskonechnye-igry/` → `articles/finite-games/`
- Папка `/admin/` удалена (не используется при архитектуре «.md → build.py → GitHub Pages»).

## Что есть в v12

- Главная — `/index.html` (+ варианты `/index-link-1..3.html`)
- Страница тега — `/tags/produkt/index.html` (пример для тега «Продукт»)
- Страница статьи — `/articles/finite-games/index.html`
- Страница 404 — `/404.html`

Структура путей уже заложена с прицелом на будущую английскую версию:
русский язык — в корне сайта, английский позже переедет в `/en/`
(`/en/articles/<slug>/`, `/en/tags/<slug>/`).

## Запуск локально

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

Чтобы увидеть связку, начни с главной, кликни по тегу «Продукт» или по заголовку
«Конечные и бесконечные игры».

## Деплой на GitHub Pages

1. Репозиторий должен быть публичный.
2. Загрузите содержимое этой папки в корень репозитория.
3. Settings → Pages → Source: ветка `main`, папка `/ (root)`.
4. Для домена `elenamasolova.com` — добавьте CNAME-файл с этим доменом
   в корень репозитория и настройте DNS по инструкции GitHub.

## Что здесь — черновик

- Тексты (биография, теги, заголовки постов, текст статьи, CTA-блок) — условные.
- Часть ссылок в верхнем меню и внутри — заглушки `#`.
- Только десктоп, только русский.
- Рабочие примеры: клик по тегу «Продукт» → страница тега; клик по заголовку
  «Конечные и бесконечные игры» → страница статьи; битая ссылка → 404.

## Палитра и шрифты

Тема «Perplexity» — тёплый светлый фон, бирюзовый акцент.

- Фон: `#fbfaf4` (светло-бежевый, «бумажный»)
- Текст: `#1f1f1e` (темно-графитовый)
- Акцент: `#20808d` (бирюзово-зелёный)
- Шрифты: IBM Plex Serif (основной), IBM Plex Sans (навигация, теги) — подключены через Google Fonts.

Изменить цвета можно в одном месте — в блоке `:root {}` в начале `styles.css`.

## История версий

- v1 — главная, тема Perplexity, теги-пилюли.

  убрана папка /admin/ (неактуальна при архитектуре «.md + build.py»).
